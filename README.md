# RelocationDevs

Plataforma de análisis y apoyo a la decisión para profesionales tecnológicos
que evalúan ciudades de reubicación internacional.

## Funcionalidades
- City Explorer — indicadores clave por ciudad
- City Comparison — comparación lado a lado
- Salary Estimator — estimación salarial por perfil
- Relocation Score — puntuación compuesta explicable

## Stack
Python · pandas · scikit-learn · Plotly · Streamlit

## Instalación
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Uso
```
python -m streamlit run app/main.py
```
